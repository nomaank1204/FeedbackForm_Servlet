# Feedback Form – Java Servlet Project

## 📌 Project Description
This project is a simple Java web application built using **Jakarta Servlet API**.  
It contains:
1. A **Feedback Form Servlet** to collect user feedback

The application demonstrates basic servlet concepts such as:
- Handling HTTP requests and responses
- Form submission using POST method
- Simple validation logic
- Dynamic HTML generation using `PrintWriter`

---

## 🛠 Technologies Used
- Java (JDK 8 or higher)
- Jakarta Servlet API
- Apache Tomcat Server
- HTML (generated inside servlet)
